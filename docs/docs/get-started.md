# 快速上手

本指南帮助你在几分钟内完成 MouseDance 的安装、授权与快捷键配置。

## 系统要求

- macOS 14 或更高版本
- 一台及以上外接显示器（单屏用户也可以使用"快捷切换"以外的功能，但多屏体验最佳）

## 1. 下载与安装

1. 前往 [Gitee Releases](https://gitee.com/qianxunclub/mouse-dance/releases) 下载最新的 `MouseDance-x.x.x.dmg`。
2. 打开 dmg，将 `MouseDance.app` 拖入"应用程序"文件夹。

::: warning 未签名应用提示
本应用未使用 Apple 开发者证书签名，首次打开时 macOS 可能提示"应用已损坏，无法打开"或"无法验证开发者"。这是未签名应用的正常提示，在终端执行以下命令解除 Gatekeeper 隔离即可：

```bash
xattr -cr /Applications/MouseDance.app
```
:::

3. 正常双击打开 MouseDance，图标将出现在屏幕右上角菜单栏。

## 2. 授予"输入监控"权限

全局快捷键监听依赖该权限，未授权时快捷键不会生效：

1. 首次启动后点击主窗口中的 **前往授权…**。
2. 在系统设置中勾选 MouseDance；若列表中已存在，请关闭后重新打开开关。

授权完成后，主窗口"输入监控权限"会显示 **已授权**。

## 3. 为屏幕录制快捷键

在主窗口 **屏幕配置** 区域：

1. 点击某块屏幕右侧的快捷键输入框。
2. 按下想要的组合键（例如 `⌘⌥1`，即 Command + Option + 1）即完成录入。
3. 再次点击输入框可重新录制。

窗口会实时显示"当前识别 N 块屏幕"，插拔显示器后列表自动更新。

## 4. 配置"快捷切换"（可选）

在 **全局配置** 中录制一个快捷键，用于在当前屏幕和上一个活跃屏幕之间快速往返。

::: tip
也可以直接快速 **双击 Command / Control / Option** 进行录入，轻量且不易与其他软件冲突。
:::

## 5. 标记屏幕（可选）

点击窗口右上角 **标记屏幕**，所有显示器上会临时叠加屏幕编号与已配置的快捷键，便于核对多屏排布。

## 6. 开启"开机自启动"（可选）

在 **全局配置** 中开启后，登录系统即自动运行，仅显示菜单栏图标，程序坞不显示图标。

## 卸载

1. 菜单栏图标 → **退出 MouseDance**。
2. 将 `/Applications/MouseDance.app` 移入废纸篓。

无其他残留文件。
